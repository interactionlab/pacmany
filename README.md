# Pac-Many
Pac-Many was developed as a reseach project.

## Setup Pac-Many
Step-by-step instructions:
* Setup a postgres database and update `config.json` accordingly
* `$ npm install update`
* `$ npm install -g nodemon`
* `$ npm start`
